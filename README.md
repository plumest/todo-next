## Requirement

- `node` version 18.17 or later.

## Getting Started

1. First, install package (if need).

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

2. Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
